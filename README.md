# Introduction 

This is a CLI program written in Rust which decodes a VIN (Vehicle identification number).

**Important:** 
- Only european VIN's get decoded. ([ISO-Norm 3779](https://www.iso.org/standard/52200.html))


## Getting Started
```
Usage: rivin <VIN>

Arguments:
  <VIN>

Options:
  -h, --help  Print help

```
